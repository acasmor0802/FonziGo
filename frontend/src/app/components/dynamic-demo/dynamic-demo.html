Demo de Manipulación del DOM con Renderer2

Este componente demuestra la creación y eliminación de elementos del DOM usando Renderer2.

Los elementos creados tienen estilos aplicados dinámicamente y responden a eventos de mouse.

+ Crear Elemento - Eliminar Elemento Limpiar Todo

Métodos usados:

  • renderer.createElement() - Crear elementos
  • renderer.appendChild() - Insertar en el DOM
  • renderer.removeChild() - Eliminar del DOM
  • renderer.setStyle() - Aplicar estilos
  • renderer.addClass() - Añadir clases CSS
  • renderer.setAttribute() - Establecer atributos
  • renderer.listen() - Añadir event listeners
